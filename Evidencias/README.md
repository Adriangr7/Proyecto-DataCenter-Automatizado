# Evidencias y Resultados

Este directorio contiene salidas reales y capturas obtenidas de la
infraestructura desplegada.

## Evidencias Incluidas
- Configuraciones en ejecución de los dispositivos
- Resúmenes BGP EVPN
- Tablas de direcciones VXLAN
- Estado de pods y servicios de Kubernetes
- Capturas de tráfico mostrando encapsulación VXLAN

Todas las evidencias corresponden a una infraestructura real y operativa.
